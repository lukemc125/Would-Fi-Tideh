# Authentic audio — manual generation guide

The free ElevenLabs tier lets you use Voice Library voices on the **website** (just not the API), so generate these by hand once and the site picks them up automatically.

## Steps

1. Open <https://elevenlabs.io> -> **Speech Synthesis** (Text to Speech).
2. Pick your **female** Jamaican voice (id `RRIjxt3K1iKEkfsLGRXU`), model **Multilingual v2**. Use the *same* voice for every clip — on the site this is the voice that reads each proverb aloud.
3. For each row: paste the **Text**, generate, **Download** the mp3, and save it as the **File** name into `audio/proverbs/`.
4. Commit the mp3s. "Daily Wud -> Hear it" and the patois lines in "Wuds of Wisdom" then use them automatically; any clip you skip falls back to the browser voice.
